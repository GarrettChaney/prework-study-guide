HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • git status: checks what branch we're currently operating in.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: loads your current work into the current branch selected on your terminal.
  • git commit -m "message": commits your changes to your currently selected branch with the appended message.
  • git pull origin main: updates your workspace to the most up-to-date version of the main branch of your repository.
